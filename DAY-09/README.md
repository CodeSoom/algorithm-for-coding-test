# DAY-09

## 힙

- 오늘의 알고리즘 유형은 **힙**입니다.
- 오랜만에 자료구조가 문제로 나왔는데요! 우선순위 큐를 구현하는 트리 자료구조예요.
- 그래서 시험에서 직접 힙을 구현하는 문제는 안나올거예요 (아마)
- 저는 파이썬으로 문제를 풀어서 힙이 구현이 되어있기 때문에 쉽게 풀었어요 ㅎㅎ
- 다른 언어를 쓰시는 분들은 힙을 구현해놓은 라이브러리가 있는지 확인해보세요! 파이팅~~
