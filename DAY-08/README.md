# DAY-08

## 그리디

- 오늘의 알고리즘 유형은 **그리디**입니다.
- 그리디는 탐욕 알고리즘인데 현재에서 가장 최선인 선택을 했을 때 해답이 나오는 알고리즘이에요.
- 매 순간 최선의 선택을 하는 것이 최선의 결과를 낸다는 보장이 되어야해서 문제가 탐욕 알고리즘이라는걸 이해하는게 중요해요.
- 그리디는 문제를 몇 개 풀어보면 감이 올거에요 ㅎㅎ 파이팅이에요!
